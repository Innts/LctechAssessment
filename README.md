# LctechAssessment
#### 專案開發規範
- .NET Core：作為專案的主要後端開發框架。 
- LINQ：用於查詢和操作數據。
- ANSI SQL：用於與DB進行交互。
- RESTful API：用於構建和維護 API 端點。
- GraphQL：用於支援靈活的數據查詢。
- JWT：用於身份驗證和授權。

#### 數據存儲和消息隊列：
- Redis：用於緩存和臨時數據存儲。
- MySQL：用於持久化的資料儲存。
- RabbitMQ 和 Kafka：用於消息傳遞和事件驅動。

#### 日誌和監控：
- Elastic Stack：用於日誌收集、分析和監控。
